== Description ==

Profile Builder - Social Connect Add-On

Extends the functionality of Profile Builder by adding social login.


NOTE:
This plugin is compatible with Profile Builder version 2.3.1 or higher


== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Social Connect tab under Profile Builder menu.
